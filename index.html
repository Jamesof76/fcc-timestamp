FreeCodeCamp: Timestamp Microservice

  1. I can pass a string as a parameter, and it will check to see if that string contains either a Unix timestamp or a natural language date (ex: January 1, 2016).
  2. If it does, it returns both the Unix timestamp and the natural language form of that date.
  3. If it does not contain a date or a Unix timestamp, it returns null for those properties.

Example Usage

https://timestamp-ms.herokuapp.com/January 1, 2016
https://timestamp-ms.herokuapp.com/1451606400000

Example Output

{ "natural": "January 1, 2016", "unix": 1451606400000 }
